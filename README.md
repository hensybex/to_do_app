Приложение для создания и трекинга задач.

На главной отображается количество активных задач и список задач

<img width="600" alt="main_page" src="https://user-images.githubusercontent.com/47056659/212928680-3cf4cea4-3636-4b66-80ca-cd9b1a868c34.png">

Задачи могут иметь разный приоритет. Задача может быть отмечена, как звершенная, тогда в списке она будет зачеркнута, либо задачу можно удалить навсегда.

<img width="600" alt="main_page_done" src="https://user-images.githubusercontent.com/47056659/212928734-b5b5a45c-bf40-4147-889f-c24882bc06b1.png">

Задачи с высоким приоритетом отображаются отдельным цветом и с дополнительной иконкой, для таких задач отображается точный дедлайн задачи.

<img width="600" alt="main_page_filled" src="https://user-images.githubusercontent.com/47056659/212928746-2db187af-1850-4577-8f9a-374b35916559.png">

Выполненные задачи можно скрыть кликом

<img width="600" alt="main_page_hidden" src="https://user-images.githubusercontent.com/47056659/212928755-e1116ed1-259d-4d55-996b-81979814ab92.png">

Добавление задач происходит на отдельной странице

<img width="600" alt="add_todo" src="https://user-images.githubusercontent.com/47056659/212928771-36b3fcbb-0a26-469c-b505-56473dabf384.png">
